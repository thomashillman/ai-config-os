# One-Liner Code Explanation

Explain the code in one clear, concise sentence that captures its essential purpose.

## Requirements

1. Single sentence maximum
2. Lead with what the code does
3. Avoid technical jargon if possible
4. Suitable for code tooltips or quick reference

## Format

[Action verb] [noun/object]: [one-liner description]

Example: "Computes the hash: applies SHA-256 to the input string and returns the hex digest."
