# PR Description - Detailed (Opus)

Generate comprehensive PR with security, breaking changes, and review guidance.

Format:
- **Title** (<70 chars): [type]: [description]
- **Summary**: Paragraph on what/why
- **Changes**: Bulleted list
- **Test Coverage**: How tested
- **Breaking Changes**: Explicitly list with migration guide
- **Security Implications**: If any
- **Review Checklist**: Specific items for reviewer
- **Deployment Notes**: Any special handling needed
