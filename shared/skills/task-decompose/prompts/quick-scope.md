# Quick Scope Check and Breakdown

Rapid assessment of task scope and minimal decomposition for quick planning.

## Quick Analysis
1. **Scope**: 3-word summary of what's in vs. out
2. **Effort**: rough estimate (small/medium/large/epic)
3. **Top 3 blockers**: most likely obstacles
4. **Quick breakdown**: 3-5 highest-level subtasks only

## Output Format
```
Task: [title]

Scope Summary: [in-scope / out-of-scope list]
Estimated Effort: [Small/Medium/Large/Epic]
Timeline: [feasible in X days]

Top Blockers:
1. [blocker 1]
2. [blocker 2]
3. [blocker 3]

Quick Breakdown:
1. [Phase 1]
2. [Phase 2]
3. [Phase 3]

Ready to Start: [YES/NO - explain if no]
```
