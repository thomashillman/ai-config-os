# Context Budget - Practical (Sonnet)

Quick decision framework:

**>50% available**: Direct approach
**25-50% available**: Consider summarization + subagents
**<25% available**: Use subagents aggressively

**When to use subagents**:
- Task is independent (research, analysis)
- Results can be summarized
- Output likely <500 tokens

Recommend 2-3 immediate actions.
